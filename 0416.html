🎯 Vue 基本練習 (04/16)

概述

這份指南整理了 Vue.js 的核心概念與指令，涵蓋了從資料渲染到使用者互動的基礎知識。

1. 聲明式渲染 (Declarative Rendering)

Vue 使用擴充自 HTML 的模板語法，根據 JavaScript 狀態來描述 HTML 樣式。當狀態改變時，HTML 會自動更新。

📌 核心概念

  • SFC (Single File Component): 將 HTML, CSS, JavaScript 封裝在 .vue 檔案中。

📌 響應式狀態 (Reactivity)

  • reactive(): 適用於物件、陣列、Map/Set。
  • ref(): 適用於任何類型，需透過 .value 存取（模板中會自動解包）。

📌 模板語法

  • Mustache 語法: {{ expression }} 用於文字內插。
💡 提示： 響應式狀態改變時，Vue 會自動追蹤依賴並重新渲染受影響的部分。

2. 屬性綁定 (Attribute Bindings)

使用 v-bind 指令將 HTML 屬性與動態值同步。

📌 基本語法

  • 完整語法: <div v-bind:id="dynamicId"></div>
  • 簡寫: <div :id="dynamicId"></div>

📌 常用屬性

  • :id - 綁定 ID
  • :class - 綁定 CSS 類名
  • :style - 綁定內聯樣式
  • :src - 綁定圖片或資源路徑
  • :href - 綁定超連結
💡 提示： 屬性綁定可以接收任何有效的 JavaScript 表達式。

3. 事件監聽器 (Event Listeners)

使用 v-on 指令監聽 DOM 事件。

📌 基本語法

  • 完整語法: <button v-on:click="increment">
  • 簡寫: <button @click="increment">

📌 常用事件

  • @click - 點擊事件
  • @submit - 表單提交
  • @input - 輸入框內容改變
  • @change - 表單元素改變
  • @mouseover - 滑鼠進入
  • @mouseleave - 滑鼠離開
  • @keydown - 按鍵按下
💡 提示： 可以傳入事件對象或自訂參數，例如 @click="handleClick($event)"。

4. 表單綁定 (Form Bindings)

Vue 提供 v-model 指令實現雙向資料綁定，是 :value 與 @input 的語法糖。

📌 基本用法

<input v-model="text">

上述代碼等同於：

<input :value="text" @input="text = $event.target.value">

📌 支援的表單元素

  • 文字輸入框: <input v-model="message">
  • 多行文本: <textarea v-model="message">
  • 核取方塊: <input type="checkbox" v-model="isChecked">
  • 單選按鈕: <input type="radio" v-model="selected">
  • 下拉選擇框: <select v-model="selected">
💡 提示： v-model 會根據表單元素的類型自動選擇正確的屬性與事件進行綁定。

5. 條件渲染 (Conditional Rendering)

根據條件決定是否渲染元素。

📌 可用指令

  • v-if: 條件為真時渲染
  • v-else: v-if 為假時渲染
  • v-else-if: 添加其他條件分支

📌 基本示例

<div v-if="seen">現在你看到我了</div>
<div v-else>看不見我</div>
⚠️ 重要特性： 當條件為假時，元素會直接從 DOM 中移除，而不只是隱藏（display: none）。

📌 v-show vs v-if

  • v-if: 真正移除/添加元素，開銷較大但隱藏狀態下不執行事件
  • v-show: 只改變 CSS display 屬性，開銷較小但元素仍在 DOM 中

6. 清單渲染 (List Rendering)

根據陣列內容迭代渲染多個元素。

📌 基本用法

<ul>
  <li v-for="item in items" :key="item.id">
    {{ item.name }}
  </li>
</ul>

📌 關鍵點

  • :key 屬性: 必須使用 :key 屬性（通常綁定唯一 ID），以協助 Vue 追蹤節點狀態並優化效能。
  • 索引訪問: 可以同時訪問索引 v-for="(item, index) in items"
  • 物件遍歷: 可以遍歷物件 v-for="(value, key) in object"

📌 陣列更新方式

  • 變更原始陣列: 使用 push(), pop(), shift(), unshift(), splice(), sort(), reverse()
  • 替換新陣列: 使用 filter(), concat(), slice() 等方法創建新陣列
  • 直接賦值: items = newArray
⚠️ 重要提示： 務必為每個列表項目提供唯一的 :key，以避免狀態混亂和效能問題。不要使用索引作為 key（尤其是在列表可能被排序或過濾時）。
