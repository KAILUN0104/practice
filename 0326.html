0326 實作內容

本週練習一：同步與非同步 (預測輸出順序)

📝 JavaScript 代碼
console.log("🐱 貓")

setTimeout(function() {
  console.log("🐶 狗")
}, 1000)

setTimeout(function() {
  console.log("🐰 兔")
}, 0)

console.log("🐟 魚")
🎯 預測輸出順序

第 1 行： 🐱 貓 (同步執行)

第 2 行： 🐟 魚 (同步執行)

第 3 行： 🐰 兔 (非同步，setTimeout 0ms)

第 4 行： 🐶 狗 (非同步，setTimeout 1000ms)

💡 解釋
  • 同步代碼： console.log("🐱 貓") 和 console.log("🐟 魚") 會立即執行
  • setTimeout 宏任務隊列： 所有同步代碼執行完後才會進入事件循環
  • setTimeout(0)： 即使延遲為 0，也會被推入任務隊列，在同步代碼之後執行
  • 執行順序優先級： 同步代碼 → 微任務隊列 → 宏任務隊列 (按時間順序)
