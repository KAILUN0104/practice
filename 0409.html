概述

本週學習內容重點是了解 Vue.js 的基本架構，並建立開發環境。我們需要安裝 NVM (Node Version Manager) 來管理 Node.js 版本， 然後使用 pnpm 和官方腳手架 create-vue 快速建立 Vue 專案。

第一步：安裝 NVM

NVM 是一個 Node 版本管理工具，允許我們快速安裝、使用及切換不同版本的 Node.js。

💡 提示： 如果您使用的是 Windows，建議安裝 nvm-windows。如果是 macOS 或 Linux，使用官方 nvm。

第二步：安裝 & 管理 Node.js 版本

使用以下指令安裝及管理 Node.js：

  • nvm install 22 : 安裝 Node.js 版本 22
  • nvm alias default 22 : 設定預設版本為 22
  • nvm list : 列出所有已安裝的 Node.js 版本
  • nvm use 22 : 切換到 Node.js 版本 22（臨時）

第三步：驗證 Node.js & npm 版本

確認安裝成功：

  • node --version : 查看 Node.js 版本
  • npm --version : 查看 npm 版本

第四步：安裝全域套件管理工具

安裝 pnpm，這是一個更快更高效的套件管理工具：

npm install -g pnpm
  • pnpm --version : 驗證 pnpm 安裝成功

第五步：建立 Vue 專案

使用官方腳手架建立新的 Vue 專案：

pnpm create vue@latest
✅ 按照提示選擇：
  • TypeScript? → 根據需求選擇 (初學者可選 No)
  • JSX Support? → 可選 No
  • Router? → 可選 Yes (了解路由功能)
  • Pinia? → 可選 Yes (狀態管理)
  • Vitest? → 可選 Yes (測試框架)
  • 其他選項 → 根據需求選擇

第六步：安裝依賴 & 啟動開發伺服器

進入專案目錄並安裝所有依賴：

cd <project-name>
pnpm install
pnpm run dev
🎯 完成！ 您應該會看到類似以下的輸出： 
  VITE v5.x.x  ready in xxx ms

  ➜  Local:   http://localhost:5173/
  ➜  press h + enter to show help

關鍵概念：什麼是 Vue.js？

  • 漸進式框架： 從簡單的交互開始，逐漸擴展到複雜應用
  • 響應式系統： 資料改變時，UI 自動更新
  • 組件化： 將 UI 拆分成可重用的組件
  • 虛擬 DOM： 高效的 DOM 操作與更新

常用 pnpm 指令

  • pnpm install : 安裝所有依賴
  • pnpm add <package-name> : 安裝新套件
  • pnpm run dev : 啟動開發伺服器
  • pnpm run build : 建立生產版本
  • pnpm run preview : 預覽生產版本
